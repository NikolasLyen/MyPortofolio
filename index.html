Tableau Excel Data Visualization SDGs

Building an interactive dashboard using Tableau to analyze elementary education gaps across 34 provinces in Indonesia (2021-2023) with 6 types of visualizations: geographic maps, bar charts, line charts, pie charts, treemaps, and scatter plots.

Data Processing:

  • Data cleaning and preprocessing on dataset with 20+ columns and 1000+ rows using Excel
  • Duplicate removal, data validation, and consistency checks
  • Created 5 dashboard stories with interactive filters

Key Insights:

  • Province with highest student-teacher ratio: Papua Pegunungan (39.89:1)
  • Dropout rate pattern: 54.75% occurs in Grade 1
  • Distribution of damaged classrooms per province
  • Analysis of school & teacher distribution, infrastructure conditions, dropout trends, and learning group participation

Spam Email Detection using Deep Learning

Python Jupyter Notebook Pandas Matplotlib Seaborn

Developing a spam email detection system using Python by comparing 5 deep learning architectures: RNN, LSTM, BiLSTM, GRU, and CNN on the SpamAssassin Public Corpus dataset (86.5% spam, 13.5% ham).

Data Analysis:

  • Exploratory data analysis with Python (Pandas, Matplotlib, Seaborn)
  • Identified feature correlation: text_length vs word_count = 0.975
  • Analysis of spam vs ham email distribution patterns

Model Performance:

  • Data preprocessing: encoding handling, text cleaning, tokenization, padding
  • Train-test split 80:20 for consistent dataset
  • Best Models: BiLSTM and CNN
  • Training accuracy: 99.64%
  • Validation accuracy: >96%
  • Validation loss: Stable <0.05
  • Outperformed RNN, LSTM, and GRU
